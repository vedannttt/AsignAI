Design a modern, clean, and professional SaaS dashboard UI for a web application titled:

"AI Agent for Automatic Assignment Creation and Checking"

The design should be suitable for both teachers and students and follow modern UI/UX principles.

--------------------------------
LAYOUT
--------------------------------

Create a full dashboard layout with:

- Left sidebar navigation
- Top navbar
- Main content area with cards and forms

--------------------------------
SIDEBAR
--------------------------------

Include menu items with icons:

- Dashboard
- Assignment Generator
- Answer Key Generator
- Upload Assignment
- Evaluation
- Plagiarism Checker
- Feedback
- Copy Detection

Style:
- Dark sidebar (blue/purple gradient)
- White icons
- Rounded hover effects

--------------------------------
TOP NAVBAR
--------------------------------

Include:
- App title: "AI Assignment Agent"
- Search bar
- User profile avatar
- Notification icon

--------------------------------
DASHBOARD PAGE
--------------------------------

Design cards for:

- Generate Assignment
- Generate Answer Key
- Upload Assignment
- Evaluate Assignment
- Plagiarism Checker
- Feedback
- Copy Detection

Each card should include:
- Icon
- Title
- Short description
- Hover effect

--------------------------------
FORM PAGES
--------------------------------

Design form layouts for:

Assignment Generator:
- Subject dropdown
- Topic input
- Difficulty selector
- Number of questions input
- Generate button

Answer Key Generator:
- Large text input
- Generate button

Upload Assignment:
- Drag and drop file upload area
- Upload progress bar

Evaluation Page:
- Upload assignment + answer key
- Evaluate button
- Results section with marks

--------------------------------
RESULT DISPLAY
--------------------------------

Design result panels showing:

- Generated text
- Feedback (strengths, weaknesses)
- Plagiarism percentage with progress bar
- Similarity score visualization

--------------------------------
STYLE GUIDE
--------------------------------

- Use modern SaaS design style
- Rounded corners (8px–16px)
- Soft shadows
- Clean typography (Poppins or Inter)
- Color palette:
  - Primary: Blue / Purple
  - Background: Light gray
  - Cards: White
- Use icons (AI, document, upload, analytics)

--------------------------------
RESPONSIVENESS
--------------------------------

Design should be responsive for:
- Desktop
- Tablet
- Mobile

--------------------------------
OUTPUT
--------------------------------

Create:
- Complete dashboard screen
- At least 3 additional screens (Assignment Generator, Upload, Evaluation)

Make the UI visually aesthetic, modern, and suitable for a hackathon demo.
